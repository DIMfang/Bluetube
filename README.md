# BlueTube
A Video-Sharing website where you can:
- Create an account
- Upload videos
- Watch your videos and other user's uploaded media
- Make comments on videos
- Like/Dislike videos

And many other features
### What was used to make this project?
- Backend
  - Java Servlets
  - Java Filters
  - HTTP Sessions
  - PostgreSQL Database
- Frontend
  - Bootstrap 4
  - FontAwesome
